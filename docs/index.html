Быстрый страт Примеры

🐝 PSR-7

Стандартные сообщениямя в формате PSR-7. Это позволяет легко интегрироваться в приложения

✨ PSR-15

Может использоваться как PSR-15 обработчик, встраиваясь в существующие приложеиня

🍿 Простота

Минимализм и простота являются ключевой особенностью микро-фреймворка

🛠 Гибкость

Микрофреймвор не навязывает архитектуру, гибкость остается на стороне разработчика

📦 Композиция

Не зависимые микро приложения могут объединяться в композицию на более высоком уровне

☝ Маршрутизация

Быстрая и гибкая маршрутизация из коробки

🤖 API

Идеально подходит для разработки REST APIs

🚀 Быстродействие

Не имеет накладных расходов и тяжелого bootstrap. Имеет почти нулевой оверхед

🎉 PHP 8 Ready

Использует передовые возможности языка PHP и ситему типов
